Authenticated Encryption with ChaCha20-Poly1305

A secure message storage demonstration utilizing Node.js, MySQL, and modern authenticated encryption for tamper-proof data security.

ChaCha20-Poly1305 is a high-speed, modern Authenticated Encryption with Associated Data (AEAD) algorithm. Originally designed by Daniel J. Bernstein and further refined for standard use by Google, it has become the gold standard for security in TLS 1.3, OpenSSH, and WireGuard. Unlike older algorithms, it provides both strong encryption (Privacy) and a cryptographic signature (Integrity) in a single, efficient pass, making it the preferred choice for mobile devices and secure modern communications.

How the Algorithm Works (Step-by-Step)

Core Terminology

1. Plaintext

The original, readable message you want to keep secret. For example: "Meeting at 5 PM."

2. Ciphertext

The encrypted, unreadable version. It looks like random characters. It is generated using a combination of the key and the nonce.

3. Nonce (Number used Once)

A unique value used for every encryption session. It ensures that the **same message encrypted twice** will look completely different, preventing attackers from recognizing patterns.

4. Authentication Tag (Auth Tag)

A special digital signature (MAC) that ensures "Integrity." If even one character of the ciphertext or nonce is changed (tempering), the auth tag will no longer match, and decryption will fail.

The Process

Step A: Encryption Flow

  • The system takes your Plaintext.
  • A random 24-digit Nonce is generated.
  • ChaCha20 uses a secret key and the nonce to scramble the data into Ciphertext.
  • Poly1305 processes the ciphertext to create a unique Auth Tag.
  • All three are saved to the database.

Step B: Decryption Flow

  • The system receives the Ciphertext, Nonce, and Auth Tag.
  • It first recalculates the Auth Tag using the secret key.
  • If it matches: The message is authentic and is converted back to Plaintext.
  • If it fails: The system identifies that the data was tampered with and rejects decryption.
